
# API de Portafolio de Inversiones

API REST para gestión de portafolios de inversión con datos en tiempo real.

## Stack
- Node.js + Express
- PostgreSQL
- JWT (access + refresh tokens)
- Zod

## Endpoints
### Auth
- POST /auth/register
- POST /auth/login
- POST /auth/refresh
- POST /auth/logout

## Setup
\`\`\`bash
npm install
npm run dev
\`\`\`
